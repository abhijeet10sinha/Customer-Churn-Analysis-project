# Customer Churn Analysis

## Project Overview

This project analyzes customer churn behavior in a telecom company using SQL and Python.

The objective of this project is to identify factors contributing to customer churn and generate business insights that can help improve customer retention.

The analysis includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights


## Dataset Information

Dataset Used:
IBM Telco Customer Churn Dataset

The dataset contains customer information such as:
- Customer demographics
- Contract type
- Monthly charges
- Payment method
- Internet service
- Customer churn status


## Tools & Technologies Used

- SQL (MySQL)

## Project Workflow

1. Data Collection
2. Data Cleaning

6. Business Insights Generation


## Key Insights

- Customers with month-to-month contracts showed the highest churn rate.
- Customers with higher monthly charges were more likely to churn.
- Customers with lower tenure had higher churn probability.
- Electronic check payment users showed higher churn behavior.
- Long-term contract customers had better retention rates.
